# Tailwind CSS Project


## Getting Started


### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/kakashijs1/TailwindProject.git
   cd TailwindProject
   cd app
   ```
2. **Install dependencies**

    ```bash
    npm install
    ```
3. **Running the Development Server**
    ```bash
    npm run dev    
    ```

your website is running at http://localhost:3000
